# Krish Zhao Kaushik — Portfolio

Personal portfolio website live at [krishkaushik.vercel.app](https://krishkaushik.vercel.app). Designed my me.

## Tech Stack

- **Next.js 14** — App Router
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** — animations
- **next-themes** — dark/light mode
- **Vercel** — deployment


## Project Structure

```
app/          # Next.js app router (layout, page, global styles)
components/   # UI and section components
data/         # Portfolio content (portfolio.ts)
public/       # Static assets (resume, profile image)
types/        # TypeScript types
```
